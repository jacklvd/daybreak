A fast job tracker for **New Grad, Entry-Level, and Internship** roles in CS,
chemistry/process/QA, and data/TPM — with **direct company apply links** (no
tracking redirects). Prefer the [live board](https://daybreak.jackvd.com) for
search and filters, or click any **Apply** link in the tables below.

**How it works**

- 🤖 A scraper hits company ATS APIs directly (Greenhouse, Lever, Ashby, Workday…) and refreshes this board **every 6 hours**.
- 🔗 Every posting links **straight to the company's own careers page** — no redirects, no trackers.
- 🌍 Only **US or remote** roles are listed; explicit non-US locations are filtered out.
- 🛂 Work authorization is inferred from posting text only. `US citizen / clearance` is flagged **only** when a posting frames it as *required* (not merely preferred).
- ⏱️ Postings older than **90 days** are dropped — no stale listings.
