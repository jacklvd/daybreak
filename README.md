## Job Board

A fast job tracker for **New Grad, Entry-Level, and Internship** roles in CS,
chemistry/process/QA, and data/TPM — with **direct company apply links** (no
tracking redirects).

- 🔎 **Browse the live board:** <CLOUDFLARE_PAGES_URL>
- 📄 Or click any **Apply** link in the tables below.

Data refreshes automatically several times a day. Every posting links straight
to the company's own careers/ATS page. Work authorization is inferred from
posting text only; citizenship/clearance is marked only when a posting frames it
as **required** (not merely preferred). Postings older than 120 days or with
explicit non-US locations are excluded.
